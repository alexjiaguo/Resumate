# Professional Resume Builder Ecosystem 📝

A high-performance, browser-based resume customization suite designed for modern professionals. This ecosystem provides a series of templates (v1.0 to v3.0) with powerful real-time customization, formatting, and content management tools.

## 🚀 Key Features

### 🏢 Multi-Layout Support
- **Resume Builder v3.0 (Latest)**: Modern sidebar layout with advanced column management.
- **Resume Builder v2.0**: Classic professional layout focusing on readability and standard hierarchies.
- **Resume Builder v1.0**: Minimalist, clean-slate design for high-impact single-page resumes.

### 🎨 Visual Customization
- **16+ Professional Presets**: Instant color schemes including *California Beaches*, *Cobalt Sky*, *Stone Path*, *Urban Loft*, and *Emerald Odyssey*.
- **Precision Typography**: Per-element font-sizing and family selection (Inter, Roboto, Open Sans, etc.).
- **Dynamic Spacing**: Granular control over page padding (Top/Bottom/Left/Right), item spacing, and section line-height (0.1 increments).

### ✍️ Intelligent Editing
- **Advanced Text Formatting**: Optimized formatting engine to prevent HTML tag nesting and style conflicts in headings.
- **Tag Insertion Tool**: Insert professional skill tags (`.tag.c`) at any cursor position with a single click.
- **Interactive Reordering**: Instantly move entire sections or individual job/education blocks up or down.
- **Headshot Management**: Quick-upload headshot tool with real-time sizing adjustments.

### 💾 Persistence & Output
- **Local Auto-Save**: All changes, content, and layout settings are automatically persisted to `localStorage`.
- **Print Optimized**: Clean CSS media queries ensure the output is pixel-perfect for PDF generation and standard A4 printing.

## 🛠 Usage Instructions

1.  **Open Template**: Launch `Resume_Builder_v3.0.html` (or any version) in a modern web browser.
2.  **Toggle Helper**: Use the **Show Page Break Marker** in the control panel to ensure your content fits on a single page.
3.  **Edit Content**: Click directly on the resume to edit text. Use the **Add Content** buttons in the panel to add sections or items.
4.  **Format**: Select text to reveal the **Text Formatting** bar for Bold/Italic/Underline and custom font sizing.
5.  **Save**: Use the **Save PDF** button at the bottom of the control panel to generate your final document.

## 📂 Project Structure

- `Resume_Builder_v2.0.html`: Classic professional template.
